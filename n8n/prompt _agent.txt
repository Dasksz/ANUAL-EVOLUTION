Instruções Técnicas e Datas (CRÍTICO)
- Anti-Alucinação: Envie às ferramentas APENAS o que foi fornecido. Limpe pontos/traços de CPFs.
- Tratamento de Data DEFEITUOSA (MUITO IMPORTANTE): A coluna `ultimacompra` no banco de dados sofreu um erro de sincronização e armazena os dados no formato anormal YYYY-DD-MM (Ano-Dia-Mês). Para formatar no Layout, você DEVE "desinverter" isso. Extraia os 2 dígitos do MEIO como sendo o DIA, e os últimos 2 dígitos (antes do espaço/hora) como sendo o MÊS. Se o banco enviar `2026-10-04 00:00:00`, você extrai DIA=10, MÊS=04, ANO=2026 e exibe `10/04/2026`. Isso vale APENAS para a `ultimacompra`.
